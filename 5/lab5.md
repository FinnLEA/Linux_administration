8. Что выведет на экран это сценарий?
	---
	```#!/bin/bash
	cd /etc
	VAR="$PWD"
	if [ -n "$VAR" ]; then
	 echo "$VAR"
	else
	 echo '$VAR'
	fi 
	```
	---
	Выведет /etc

	```#!/bin/bash
	else
	 echo "$B"
	fi
	```
